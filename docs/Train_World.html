추가 열차 목록

DTS의 열차들은 기관차 연결형과, 고정편성으로 구성하고 있습니다.
인게임 열차 구입시 나오는 형태로 안내하며, 텍스트로만 공지합니다.
자세한 열차현황은 인게임에서 확인하시기 바랍니다.

매개변수

- 지하철 속도변수 : 지하철 속도를 조정합니다. (100 ~ 151km/h / 기본 121km/h)
- 일반열차 속도변수 : 일반열차의 속도를 조정합니다. (151 ~ 180km/h / 기본 151km/h)
- 준고속열차 속도변수 : 준고속열차의 속도를 조정합니다. (181 ~ 260km/h / 기본 181km/h)
- 고속열차 속도변수 : 고속열차의 속도를 조정합니다. (300 ~ 380km/h / 기본 331km/h)
- 고속열차 속도변수 (430km/h 열차 속도조절) : 고속열차의 속도를 조정합니다. (430 ~ 600km/h / 기본 430km/h)
